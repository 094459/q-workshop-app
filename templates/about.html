How It Works

Our polling application allows users to create, vote on, and view results of various polls. Here's how it works:

  1. Register: New users can create an account to start creating polls.
  2. Login: Registered users can log in to access all features.
  3. Create Polls: Logged-in users can create new polls with multiple options.
  4. Vote: Anyone can vote on existing polls (one vote per IP address).
  5. View Results: After voting, users can see the current results of the poll.
